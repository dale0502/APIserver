# 【作業】實作具電商功能 APIserver 
***
## 根據課程影片描述，實作出一個 API 應用程式，並具有基本電商的購物和會員功能。

### 一、功能列表:

1. Cart 購物車功能，可以建立、讀取、編輯( 添加產品 )和移除產品

2. Auth 會員註冊、登入與登出功能

3. 結帳功能 + VIP 優惠 + 商品數量檢查防呆



### 二、系統功能:

1. 需使用 Eloquent Model 完成功能

2. 需使用軟刪除特性

3. 需有資料 Validator

4. 需有資料預產生器 Seeder
